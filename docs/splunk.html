Splunk User Documentation

Splunk is a powerful platform used for searching, monitoring, and analyzing machine-generated big data via a web-style interface. It helps you visualize and make sense of data from various sources such as application logs, system metrics, and network traffic.

  • Open your browser and visit: https://splunk.backyardbandwidth.com:8000
  • Enter your username and password provided at time of purchase.
  • Click Login to access your dashboard.

Ingestion means collecting and storing data from sources like logs or traffic.

  • Data is broken into events and stored in indexes.
  • Only ingested data is searchable or visualized in Splunk.
  • Go to the Search & Reporting app.
  • Run a query like:
    • index=[PROVIDED INDEX NAME] [SEARCH QUERY]
  • Example:
    • index=abc application
  • Admins configure source ingestion.
  • To limit search results:
    • index=main | head 100
  • This shows the first 100 events.
  • Run a search in the Search & Reporting app.
  • Click "Save As" → "Dashboard Panel".
  • Name it and choose/create a dashboard.
  • Click Save.
