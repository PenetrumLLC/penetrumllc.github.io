VPN Documentation

Visit this link to purchase and download a VPN profile from the Backyard Bandwidth website.

  • Windows: Download from OpenVPN or use our VPN GUI.
  • Linux: Install with: 
    sudo apt install openvpn
    or 
    sudo yum install openvpn
  • Mac: brew install openvpn
  • Android/iOS: Use OpenVPN Connect.
  • Windows: Place the `.ovpn` file in: 
    C:\Program Files\OpenVPN\config\
  • Linux/Mac: 
    sudo openvpn --config /path/to/your-profile.ovpn
  • Android/iOS: Import into OpenVPN Connect.
  • Windows: Right-click the GUI and select "Connect."
  • Linux/Mac: 
    sudo openvpn --config /path/to/your-profile.ovpn
  • Android/iOS: Tap "Connect" in the app.

Still Need Help?

Provide the following when contacting support:

  • Your OS and version (e.g., Windows 11, Ubuntu 22.04).
  • Version of OpenVPN or VPN client used.
  • A detailed description of the problem.
  • Any error messages you received.
