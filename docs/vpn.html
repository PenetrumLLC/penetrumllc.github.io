VPN Documentation

1. Downloading the VPN Profile

Visit this link to purchase and download a VPN profile from the Backyard Bandwidth website.

2. Installing OpenVPN
  • Windows: Download and install OpenVPN or use our experimental Backyard Bandwidth VPN GUI.
  • Linux: Install OpenVPN with: sudo apt install openvpn (Debian/Ubuntu) or sudo yum install openvpn (CentOS).
  • Mac: Install using Homebrew: brew install openvpn.
  • Android/iOS: Use OpenVPN Connect (Android) or OpenVPN Connect (iOS).
3. Configuring the VPN Client
  • Windows: Place the .ovpn file in C:\Program Files\OpenVPN\config\ and launch OpenVPN GUI or use our Backyard Bandwidth VPN GUI.
  • Linux/Mac: Run sudo openvpn --config /path/to/your-profile.ovpn.
  • Android/iOS: Import the .ovpn profile into OpenVPN Connect.
4. Connecting to the VPN
  • Windows: Right-click OpenVPN GUI or Backyard Bandwidth VPN GUI and select "Connect."
  • Linux/Mac: Run sudo openvpn --config /path/to/your-profile.ovpn.
  • Android/iOS: Open the OpenVPN app and tap "Connect."
